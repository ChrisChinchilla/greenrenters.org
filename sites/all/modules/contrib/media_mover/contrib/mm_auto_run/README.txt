
Media Mover Auto Run
--------------------------------------------------------------------
This module fills in the gap between running Media Mover configurations
on cron jobs and converting items attached to Drupal nodes when they 
are inserted or updated. 

You can configure which of your configurations will be run on a specific
node when it is inserted or updated.

Installation
--------------------------------------------------------------------
 * place module in your sites/site_name/modules directory or 
   sites/all/modules
 * enable module at admin > build > modules
 * configure module at admin > settings > media mover auto run
